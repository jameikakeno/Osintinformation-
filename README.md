🕵️ Keneviz - OSINT Information Tool

🌟Bilgi : Keneviz — Telefon numarası, domain, kullanıcı adı, IP adresi, MAC adresi ve veritabanlarında arama yapmak için geliştirilmiş güçlü bir OSINT (Açık Kaynak İstihbarat) aracıdır. Çoklu platform desteği ile hızlı ve etkili bilgi toplama sağlar.

Bu araç, Termux, Linux, Windows ve tüm ana terminal ortamlarıyla uyumludur.

---

⚡️ Özellikler

· Telefon numarası sorgulama (ülke, operatör, saat dilimi)
· Domain/Web sitesi whois sorgulama
· Kullanıcı adı sorgulama (12+ platform)
· IP adresi konum sorgulama
· Veritabanında metin arama
· MAC adresi üretici sorgulama
· Renkli terminal çıktısı
· Hızlı ve güvenilir

---

🖥 Kurulum

git clone https://github.com/jameikakeno/OSINT-Tool.git
cd OSINT-Tool
pip install phonenumbers python-whois requests
python demon.py

Adım | İşlem
